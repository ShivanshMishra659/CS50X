## рҹӣ пёҸ How to Run

### рҹ”ө C Programs вҖ” Weeks 1 to 5

Designed for the [CS50 Codespace](https://cs50.dev/), which includes `cs50.h` and `make` out of the box. To compile locally:

```bash

make solution

# Or using clang directly
clang -o solution solution.c -lcs50

# Run
./solution
```
